前言

轉眼間，也在 655 打滾了兩年多了
但在準備今年的 Promote Presentation 卻發現似乎沒有什麼好寫的（崩潰

看著周圍的人（包含實習生）不斷的成長，總覺得自己好像一直停滯不前…

所以在年假期間，開始嘗試一些新的東西，這個部落格就是其中一樣

過往

以往試過了不少的部落格服務，但因為懶癌發作都沒有持續經營下去

  • Yahoo 奇摩部落格 （已關站）
    • 以前玩楓之谷弄的一個部落格，沒記錯的話XD
  • 無名小站 （已關站）
    • 國中時最流行的一個部落格服務，幾乎人手一個，黑歷史滿滿的一個時期，曾透過 Web Archive 回去看，我到底都寫了些什麼啊（炸
  • Blogger
    • 當初好像是拿來放些技術的東西，但是…沒有動力繼續經營下去，就這樣荒廢了
  • Logdown
    • 衝著可以用 markdown 就拿來用了，但後來也是沒有持續經營

現在

一方面是希望自己把在工作上學習到的東西記錄下來，另一方面是覺得腦容量已經不太足夠了，希望有個地方能夠代為保存，就決定來弄個部落格了

部落格系統採用 Hexo 並搭配佈景主題 hexo-theme-material

至於會選擇 Hexo 主要的原因還是，可以直接放 GitHub Page 上，不用另外找主機放，比較不用煩惱主機維運的問題。

寫到這邊，期許自己未來能持之以恆，找回最初的熱情…
